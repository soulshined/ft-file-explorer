Options

Property Description Values
allowKeyboardEventsOnFocus Allows users to use pre-determined keyboard shortcuts while they are navigating the tree with arrow keys and on focus <boolean>true
buttons.*.text Customize the text for a button listed below <string>Button Name
buttons.addFile.visible Include the 'Add File' button for a given explorer tree <boolean>true
buttons.addFolder.visible Include the 'Add Folder' button for a given explorer tree <boolean>true
buttons.delete.visible Include the 'Delete' button for a given explorer tree <boolean>true
expandFolders Enables/disables if folders are expanded on load <boolean>false
explorerStyle Shows the explorer in one of allowed styles <string>material
rootPath Set the first 'file' that displays on load. If omitted, the first file found will be selected <string>null
